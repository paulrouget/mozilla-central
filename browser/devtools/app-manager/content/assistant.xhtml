Welcome to the App Manager. This tool will help you build and install apps for Html web apps compatible devices (like Firefox OS).

To be able to debug web applications, it's recommended to use a device (like a Firefox OS phone) or install the Firefox OS simulator. This assistant will help you setup your device or install the simulator.

  • Setup a device
  • Install a simulator

Setup a device

To connect a device, it needs to be plugged to your computer via USB. It also needs to be linked to your computer via a tool named adb. Firefox can handle the adb connection for you (no need to install adb), or you can do it manually (install and start adb yourself).

  • Let Firefox control adb
  • Setup adb manually
